FAQ

+

How does text as file module work?

Just type or copy and paste the text in the space provided and then add the filename, then choose the file extension from the provided options and click on the 'Save as File'.It is as simple as that.

+

How does speech to text module work?

Just type or copy and paste the text or use 'Start recognition' button to record the voice and use 'Pause Recognition' to pause or stop recording the voice, instantly you can see the text which is recorded in the space provided, you can also check for corrections if necessary and finally, 'Save Notes' for future referance.It is also available in speech format.

+

How does text to speech module work?

Just type or copy and paste the text in the space provided and select voice from the options provided then use the 'Convert To Speech' button to listen to the speech format of text provided.

+

how does translate module work?

Language translator is a program that is used to translate text into multiple languages like French, Hindi, Spanish, etc. In our language translator, users can easily translate text into different languages, copy translated text, and convert text to speech. It is similar to Google Translate.
